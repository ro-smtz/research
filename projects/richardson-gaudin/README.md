# Richardson-Gaudin model

Python implementation of Richardson-Gaudin exactly solvable models and their application to quantum chemical systems, based on my Master's thesis work.

**Language:** Python (NumPy, SciPy)  
**Status:** Planned — reimplementation of thesis work

## Physics

Richardson-Gaudin models are a family of exactly solvable many-body Hamiltonians built from $\mathfrak{su}(2)$ algebras. The eigenstates are determined by a set of rapidities $\{e_\alpha\}$ satisfying the Richardson equations:

$$\sum_{j=1}^{N} \frac{1}{\epsilon_j - e_\alpha} - \sum_{\beta \neq \alpha} \frac{2}{e_\beta - e_\alpha} = \frac{1}{g}$$

**Reference:** Master's thesis — *Aplicación del método Richardson-Gaudin a sistemas químicos*,  
Universidad Veracruzana (2021–2023). Advisor: Dr. Sergio A. Lerma-Hernández.
