# matlab-tools

MATLAB scripts and functions used for data acquisition, processing, and analysis in the Advanced Nanophotonics Laboratory.

## Structure

Add scripts here organized by technique or experiment. Suggested subfolders as the directory grows:

```
matlab-tools/
├── fourier-microscopy/     # Fourier-space image processing
├── spectroscopy/           # Angle-resolved and time-resolved spectra
└── utils/                  # Shared helper functions
```

## Notes

- Each script should include a header comment with purpose, inputs, outputs, and dependencies.
- Data files (.mat, large datasets) are not tracked here — use a shared lab drive or Zenodo for raw data.
